# Simplemenu
Simple menu test project.

## Connections

| Display (ST7920) | Huzzah Feather ESP8266 |
|------------------|------------------------|
| VCC              | VUSB                   |
| GND              | GND                    |
| RS (CS)          | GPIO 2                 |
| R/W              | GPIO 13 (MO)           |
| E                | GPIO 13 (SCK)          |
| PSB              | GND                    |
| BLA              | VUSB                   |
| BLK              | GND                    |
